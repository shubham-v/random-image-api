# random-image-api
A simple api for viewing random images on browser each linked to an id.

# Local
git clone https://github.com/shubham-v/random-image-api.git                                                                                                         
cd random-image-api                                                                                                                                          
./gradlew bootRun                                                                                                                                                        
curl localhost:8080/images                                                                                                                                          
wget localhost:8080/1                                                                                                                           
wget localhost:8080/2                                                                                                                               
wget localhost:8080/1                                                                                                                     
curl localhost:8080/images
